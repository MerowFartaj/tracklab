# Roadmap

This roadmap is intentionally scoped around getting a stable DAW foundation before adding broader platform support.

## v0.1, Foundation, Weeks 1-6

- Tracktion Engine integrated
- Audio playback works
- Basic timeline UI with audio clip import
- Project save and load
- Single track type, audio only

## v1.0, Target Launch, Months 5-7

- Audio and MIDI tracks
- VST3 and AU plugin hosting
- Modern mixer with volume, pan, mute, solo, and sends
- Custom-built piano roll
- Connector framework
- 4-5 launch connectors: Splice, Local Library, Spotify, Google Drive, Suno
- AI co-producer with tool use across connectors
- Camelot wheel and BPM compatibility deterministic layer
- macOS only

## v1.1, 1-2 Months After v1.0

- Automation lanes
- Audio recording from input
- Send and return effects expanded
- More connectors, including Loopcloud, Tracklib, and Hugging Face
- AI mix suggestions

## v2, Future

- Windows support
- Linux support
- More AI capabilities
- Collaboration features
